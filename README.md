# Throttle Function

Implementación de una función `throttle` en TypeScript, que limita la frecuencia de ejecución de una función en respuesta a eventos repetidos.

## Función `throttle`

La función `throttle` asegura que una función no sea ejecutada más de una vez en un intervalo de tiempo específico, incluso si la función es llamada repetidamente en un período corto.
